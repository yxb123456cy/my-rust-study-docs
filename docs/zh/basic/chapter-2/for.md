# Rust中的 for 循环

`for` 循环是 Rust 中最常用、最安全、也是最高效的循环构造。它主要用于遍历集合（如数组、元组、迭代器）中的元素，或者执行指定次数的循环。

## 遍历集合

在 Rust 中遍历数组或其它集合时，强烈推荐使用 `for` 循环，而不是 `while` 循环。

```rust
fn main() {
    let a = [10, 20, 30, 40, 50];

    // 遍历数组 a 中的每一个元素
    for element in a {
        println!("当前的值是: {}", element);
    }
}
```

使用 `for` 循环的好处：
1. **安全性**：不需要手动管理索引，彻底消除了数组越界 Panic 的风险。
2. **简洁性**：代码更加清晰易读。
3. **高性能**：因为遍历是确定的，编译器无需在每次迭代中都插入边界检查，从而提升了运行效率。

## 使用 Range 执行特定次数的循环

如果你只是想将某段代码执行特定的次数，可以使用 Rust 标准库提供的 **`Range`** 类型，结合 `for` 循环来实现。

`Range` 可以通过 `start..end` 语法生成一个包含 `start` 但不包含 `end` 的数字序列（左闭右开区间）。

```rust
fn main() {
    // 打印从 1 到 3 的数字（不包含 4）
    for number in 1..4 {
        println!("{}!", number);
    }
    println!("发射！");
}
```

### 反转序列：rev()

你可以通过调用 `.rev()` 方法来反转 `Range` 序列：

```rust
fn main() {
    // 打印 3, 2, 1
    for number in (1..4).rev() {
        println!("{}!", number);
    }
    println!("发射！");
}
```

### 包含结束值的范围

如果你希望区间包含结束值，可以使用 `..=` 语法（左闭右闭区间）：

```rust
fn main() {
    // 打印 1, 2, 3, 4
    for number in 1..=4 {
        println!("{}!", number);
    }
}
```

## 总结

- **`for`**：首选的循环方式，安全且高效，适合遍历集合或指定次数的循环。
- **`while`**：适合当需要基于某种条件表达式为 true 时重复执行的场景。
- **`loop`**：适合需要无限循环，或者需要从循环中返回值的场景。
