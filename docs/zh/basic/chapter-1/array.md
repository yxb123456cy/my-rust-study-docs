# Rust的复合类型 (二)：数组 (Array)

另一种包含多个值的方式是**数组 (Array)**。与元组不同，数组的两个最核心特性是：

1. **类型必须相同**：数组中的每个元素都必须是相同的类型。
2. **长度固定**：在 Rust 中，数组一旦声明，它们的长度也是固定的，不能增长或缩小。

如果你需要一个可以动态增长或缩小的集合，你应该使用 `Vector`（向量），这将在后续章节详细讨论。当你确定元素的数量不需要改变时，数组是非常有用的。

## 1. 定义数组

你可以把一系列的值用逗号分隔后放在**方括号 `[]`** 中来创建一个数组。

```rust
fn main() {
    // 自动推断类型
    let a = [1, 2, 3, 4, 5];
    
    // 一些常见的数组使用场景：月份
    let months = ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"];
}
```

### 显式指定类型和长度

当你需要明确指定数组的类型时，使用方括号，在里面写上**元素的类型**，一个**分号**，接着是**数组的长度**。

```rust
fn main() {
    // a 是一个包含 5 个 i32 类型元素的数组
    let a: [i32; 5] = [1, 2, 3, 4, 5];
}
```

### 初始化包含相同值的数组

如果你想要创建一个包含相同元素的数组，你可以指定**初始值**，然后是一个**分号**，接着是**数组的长度**。

```rust
fn main() {
    // 创建一个包含 5 个元素，且每个元素都是 3 的数组
    let a = [3; 5]; 
    
    // 上面的代码完全等价于：
    // let a = [3, 3, 3, 3, 3];
}
```

## 2. 访问数组元素

因为数组在内存中是一块连续分配的固定大小的空间（分配在栈上），你可以使用**方括号索引**来访问数组中的元素。索引从 `0` 开始。

```rust
fn main() {
    let a = [10, 20, 30, 40, 50];

    let first = a[0]; // 获取第一个元素，值为 10
    let second = a[1]; // 获取第二个元素，值为 20
    
    println!("第一个元素: {}, 第二个元素: {}", first, second);
}
```

## 3. 无效的数组元素访问 (越界)

如果你尝试访问超过数组末尾的元素，会发生什么？

```rust
fn main() {
    let a = [1, 2, 3, 4, 5];

    // 尝试访问第 10 个元素（索引为 9）
    let index = 9;
    let element = a[index]; 
    
    println!("元素是: {}", element);
}
```

如果你运行这段代码，程序会发生 **Panic (恐慌/崩溃)**，并抛出类似以下的运行时错误：

```text
thread 'main' panicked at 'index out of bounds: the len is 5 but the index is 9'
```

**这是 Rust 内存安全原则的一个体现**：
在许多底层语言中，这种检查是不会被执行的，当提供了一个不正确的索引时，可以访问到无效的内存（这会导致非常严重的安全漏洞）。而 Rust 会在运行时（或编译时，如果能检测到）通过立即退出程序来保护你不受到这类错误的伤害，从而防止内存访问越界。
