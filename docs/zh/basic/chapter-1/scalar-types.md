# Rust的标量类型 (基本数据类型)

在 Rust 中，每一个值都属于某一种 **数据类型 (Data Type)**，这告诉 Rust 应该如何处理这些数据。由于 Rust 是一门**静态类型语言**，这意味着它在编译时就必须知道所有变量的类型。

Rust 中的数据类型主要分为两类：**标量类型 (Scalar Types)** 和 **复合类型 (Compound Types)**。本篇文档将重点介绍标量类型。

**标量类型**代表一个单独的值。Rust 拥有四种基本的标量类型：**整数**、**浮点数**、**布尔值**和**字符**。

---

## 1. 整数类型 (Integer Types)

整数是没有小数部分的数字。Rust 提供了多种内置的整数类型，主要区别在于占用的内存大小（位数）和是否有符号（能否表示负数）。

| 长度 | 有符号 (Signed) | 无符号 (Unsigned) |
| --- | --- | --- |
| 8-bit | `i8` | `u8` |
| 16-bit | `i16` | `u16` |
| 32-bit | `i32` | `u32` |
| 64-bit | `i64` | `u64` |
| 128-bit | `i128` | `u128` |
| arch (架构相关) | `isize` | `usize` |

- **有符号 (`i`开头)**：可以表示正数、负数和零。例如 `i8` 的范围是 -(2^7) 到 2^7 - 1。
- **无符号 (`u`开头)**：只能表示零和正数。例如 `u8` 的范围是 0 到 2^8 - 1 (即 0 到 255)。
- **架构相关 (`isize` 和 `usize`)**：取决于你的程序运行的计算机架构。在 64 位架构上就是 64 位的，在 32 位架构上就是 32 位的。通常用于集合的索引。

**默认类型**：如果在定义时不明确指定，Rust 默认的整数类型是 `i32`，因为它的执行速度通常是最快的。

```rust
fn main() {
    let a: i32 = -100; // 显式指定 i32
    let b: u32 = 250;  // 显式指定 u32
    let c = 100;       // 默认推断为 i32
    
    // 也可以在数字后面带上类型后缀
    let d = 57u8;
    
    // 为了可读性，Rust 允许在数字字面量中使用下划线 `_` 
    let big_number = 1_000_000;
    
    println!("a = {}, b = {}, c = {}, d = {}, big = {}", a, b, c, d, big_number);
}
```

## 2. 浮点类型 (Floating-Point Types)

浮点数是带有小数部分的数字。Rust 有两种基础的浮点数类型：

- `f32`：单精度浮点数（占据 32 位）。
- `f64`：双精度浮点数（占据 64 位）。

**默认类型**：在现代 CPU 中，`f64` 的速度与 `f32` 几乎相同，但精度更高，因此 Rust 的**默认浮点类型是 `f64`**。

```rust
fn main() {
    let x = 2.0; // 默认推断为 f64
    let y: f32 = 3.0; // 显式指定为 f32

    println!("x = {}, y = {}", x, y);
}
```

---

## 3. 布尔类型 (Boolean Type)

布尔类型代表的是“逻辑值”，它只有两个可能的值：`true` (真) 和 `false` (假)。在 Rust 中，布尔类型使用 `bool` 关键字表示，它在内存中只占用一个字节。

布尔值最常用于控制流代码（如 `if` 表达式）中。

```rust
fn main() {
    let is_learning_rust = true;
    let is_give_up: bool = false; // 显式指定类型

    if is_learning_rust {
        println!("继续加油！");
    }
}
```

## 4. 字符类型 (Character Type)

在 Rust 中，`char` 类型是语言中最原生的字母类型。它用于表示单个字符。

**关键特性**：
- 使用**单引号** `' '` 来声明（字符串使用的是双引号 `" "`）。
- Rust 的 `char` 类型大小为 **4 个字节 (4 bytes)**。
- 它代表的是一个 **Unicode 标量值 (Unicode Scalar Value)**。这意味着它可以表示的不仅仅是 ASCII 字符，还包括带重音的字母、中文、日文、韩文、甚至是 Emoji 表情符号！

```rust
fn main() {
    let c = 'z';
    let z: char = 'ℤ'; // 带有显式类型注解
    let heart_eyed_cat = '😻'; // 支持 Emoji 表情

    println!("字符: {}, {}, {}", c, z, heart_eyed_cat);
}
```

> **提示**：在 Rust 中，字符串 (String) 底层是由字节集合构成的，这与单纯的字符数组并不一样。我们会在后续章节中专门探讨复合类型和字符串。
